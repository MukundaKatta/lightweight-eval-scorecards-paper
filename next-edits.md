# Recommended Next Edits

## Before public posting

1. Do one accuracy pass on the bibliography against the exact venue style you choose
2. Decide whether to keep the paper as a technical report or tighten it into a shorter position-style preprint
3. Add one small workflow figure
4. Add one table of review dimensions and example questions
5. Add a short appendix that maps each public artifact to the paper's workflow stages

## Best first posting route

If the goal is speed and credibility, publish this package to Zenodo first.

## Best second route

After the Zenodo DOI exists, post the manuscript to OSF Preprints or SSRN.

## Best later route

Adapt the manuscript for arXiv when:

- the references have been checked against the final venue format
- the formatting is cleaned up
- you are comfortable with the endorsement requirement
